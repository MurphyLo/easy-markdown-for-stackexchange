# Easy Markdown for StackExchange

更方便地对 StackExchange 上的回答进行摘取，从而

- 整理积累到自己的 Markdown 编辑器中
- 炫 ChatGPT 嘴里进一步分析

# 安装
需要浏览器装有 [Tampermonkey](https://tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/) 插件, 下方表格中挑一个链接安装.
